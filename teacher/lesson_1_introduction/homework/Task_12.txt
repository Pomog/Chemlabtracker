Самостоятельно или при помощи преподавателя
создать в репозитории на GitHub Kanban Board
для управления задачами вашей команды.

Для создания Kanban Board на GitHub нужно находясь
на главной странице репозитория перейти в раздел Projects.
Находясь в разделе Projects нажмите зелёную кнопку New Project,
дайте название вашему проекту (такое же как и название модуля
в репозитории), в разделе Project Template выберите Basic Kanban.

Альтернативой для GitHub Kanban Board может служить Trello.
Обязательно исследуйте эту альтернативу, она вам может понравиться:
https://trello.com/
